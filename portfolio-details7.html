League of legend wiki

The web application built with Spring Boot and Angular showcases the champions of League of Legends along with their statistics and abilities. It leverages the power of Spring Boot to handle the backend logic and data management, while Angular provides a dynamic and responsive user interface for users to explore the champions' details.

Tasks:

  • Development of new features to enhance champion display and interaction
  • Integration of League of Legends API to fetch champion data
  • Designing and implementing responsive user interfaces using Angular
  • Handling user interactions and displaying champion statistics
  • Implementing search and filtering functionality for champions
  • Optimizing performance and loading times for a seamless user experience
  • Testing the application for functional correctness and usability
  • Deployment of the application using Spring Boot and Angular build tools

Technologies:

Spring Boot, Angular, DataDragon API, HTML, CSS, TypeScript, RESTful API, JSON, Git, IntelliJ IDEA
